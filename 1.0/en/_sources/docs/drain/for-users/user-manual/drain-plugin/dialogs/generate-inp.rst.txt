===================
Dialog Generate INP
===================

.. only:: html

    .. contents::
       :local:

Tool that allows export the network to an inp file.

.. figure:: img/generate-inp.png

  Window to generate the inp file.

By clicking on the *Options* button will open a window where we can set the inp export options.

Then we must specify a name and a path for the file by clicking ont the button with three dots.
