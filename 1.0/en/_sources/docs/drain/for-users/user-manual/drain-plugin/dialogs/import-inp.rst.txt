=================
Dialog Import INP
=================

.. only:: html

    .. contents::
       :local:

Tool for import the network from an inp file.

.. figure:: img/import-inp.png

  Window for import an inp file.

By clicking in the button with three dots, we open an explorer to search the inp file.
Then, clicking in *OK* button will start the import.
